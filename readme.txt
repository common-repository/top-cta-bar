Add a full width top most bar upon header area by creating a campaign to collect email.

== Description ==

A simple plugin that adds a full width top bar region above the default header by adding text with custom color and size. Customise the display by adding a background colors via the plugin option.

This plugin will support any theme to be installed.

== Installation ==

1. Upload 'top-cta-bar' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the display from the plugin settings option.
3. Done!

== Frequently Asked Questions ==

= I installed the plugin but cannot see the top bar =

This plugin will only work when you set any campaign from option and activate that campaign. And you will get all registered emails from there
